    Dictionary Info

    📖

    In-app dictionary

    Uses Wiktionary — free, no account needed. Coverage is best for common words. If a word isn't found, try its base (dictionary) form.

    🤖

    Richer lookups via MCP server

    A local MCP server is included in docsync-mcp/. Once running, ask Claude Code directly:
    "Look up 사랑" or "What does 감사합니다 mean?"
    Claude will look up the word and explain it in context.

    To start it: open a terminal in docsync-mcp/, run npm install then add it to Claude Code settings.
